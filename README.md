# Python Data Analysis Project

## Project Overview
This project focuses on exploratory data analysis (EDA) using Python.

The dataset contains salary-related information and was analyzed to extract meaningful insights.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Jupyter Notebook

## Project Steps
1. Data loading
2. Data cleaning
3. Exploratory Data Analysis
4. Data visualization
5. Insight extraction

## Key Insights
- Salary distribution patterns were analyzed.
- Relationship between experience and salary was examined.
- Important trends were identified through visualization.

## Files in This Repository
- `Data_Analysis.ipynb` → Main analysis notebook
- `salaries_clean.csv` → Cleaned dataset
